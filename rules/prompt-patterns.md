# 提示词规律

## 通用公式

```text
主体 + 动作 + 场景 + 风格 + 构图 + 光影 + 材质细节 + 色彩 + 质量增强 + 负面约束
```

## 关键维度

### 主体

说明画面里最重要的人、物或角色，优先明确：身份、年龄感、服装、发型、表情、姿态、核心特征。

### 场景

说明地点、时代、季节、天气、环境元素和空间层次。场景要服务主体，不要堆砌无关元素。

### 风格

记录稳定可复用的风格词，例如：电影感、写实摄影、国风插画、赛博朋克、黏土质感、3D 盲盒、儿童绘本、水彩、像素艺术。

### 构图与镜头

优先指定景别、视角和构图方式，例如：半身像、全身像、特写、低角度、俯视、中心构图、三分法、浅景深。

### 光影

光影决定质感和情绪。常用表达：柔和自然光、逆光、伦勃朗光、霓虹光、黄金时刻、体积光、电影级布光。

### 材质与细节

材质词能显著提升画面可控性，例如：丝绸、皮革、金属、玻璃、陶瓷、毛绒、半透明、磨砂、颗粒感。

### 负面约束

负向提示词用于减少常见错误。人物类优先限制手指、肢体、五官、比例、重复主体；商业海报类优先限制水印、logo、乱码文字。

## 输出原则

- 简单需求不要原样扩写，要补足视觉信息。
- 如果需求缺少风格，默认给出 1 个主推版本，并附 2 个可替换风格方向。
- 提示词要可复制，避免解释性废话混入提示词正文。
- 中文用户需求优先输出中文提示词；如平台更适合英文，可额外给英文版。
