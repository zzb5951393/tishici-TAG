# 基于单体建筑参考图补全城市街景

## 适用场景

当参考图是单栋 2D 游戏建筑，例如店铺、餐馆、民居、商店招牌，需要基于原图视角、建筑结构、轮廓关系和街区布局，扩展为完整城市街景、商业街或街区地图时，使用本案例。注意：场景类提示词默认不要输出像素风格，即使参考图是像素图，也只参考视角、结构、配色和布局，不继承像素画风。

## 参考图内容分析

- 主体：一栋肉铺 / 小店建筑，正面招牌写有 `BUTCHER SHOP` 和中文“猪肉”，招牌上有可爱猪脸图案。
- 结构：窄高型多层建筑，底层店面，二层大招牌，上方有斜屋顶和后方更高的方形楼体。
- 视角：传统 2D 游戏建筑视角，建筑正面角度正交俯视 / 轻微俯视，只看到屋顶和一个正面立面，不看到建筑侧面，不是写实透视。
- 风格处理：不要继承像素画风；只保留灰紫 / 米灰 / 蓝紫配色关系、可爱小店气质和游戏城市场景感，转为高完成度 2D 手绘 / 手游场景风。
- 可扩展元素：街道、人行道、相邻店铺、路灯、电线杆、垃圾桶、自行车、招牌、遮阳棚、路边摊、树池、斑马线、城市杂物。

## 生成原则

1. 保留参考图的建筑正面角度正交俯视游戏建筑视角、建筑结构、招牌位置、窄高比例和配色关系，但不要保留像素风格。
2. 不要只复制单栋建筑；要基于这栋肉铺向左右和前方补全完整城市街景。
3. 建筑可作为街景中的主店铺，周围补充相邻小店、道路、人行道和城市道具。
4. 如果要保留原图文字，需明确写“保留 BUTCHER SHOP 和 猪肉”；如果不需要精确文字，应写“不要乱码文字，招牌可用简化图形或清晰短字”。
5. 避免变成写实城市、电影镜头、强透视街拍或纯平面正视图。

